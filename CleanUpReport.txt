## Kalan Uyarılar

Projede hala 172 uyarı bulunmaktadır. Bunlar genellikle şu kategorilerdedir:

1. **Nullable referans türleri uyarıları (CS8632)**: nullable reference types için uygun bağlam olmadan nullable annotation kullanımı
2. **Non-nullable özellik uyarıları (CS8618)**: Constructor'dan çıkarken non-nullable özelliklere değer atanmamış
3. **Async metot uyarıları (CS1998)**: await operatörü kullanılmayan async metotlar
4. **SQL injection uyarıları (EF1002)**: SQL sorgularında olası SQL injection riski
5. **Paket uyumluluğu uyarıları**: BouncyCastle ve iTextSharp paketlerinin .NET 8.0 ile uyumluluğu

## Öneriler

1. Nullable reference type uyarılarını gidermek için proje dosyasında Nullable enable etmek ve gerekli yerlere ? eklemek
2. Non-nullable property'leri ya nullable yapmak ya da constructor'da başlangıç değerleri atamak
3. Async metotlarda await kullanmak veya async keyword'ünü kaldırmak
4. SQL sorguları için parametreli sorgular kullanmak
5. Eski paketleri güncel versiyonlarıyla değiştirmek (BouncyCastle ve iTextSharp)
